# MarBou Extensions Repository

A Mihon/Tachiyomi extension repository hosting the **Comix.to** extension.

## Add to Mihon

1. Open Mihon
2. Go to **More → Settings → Browse → Extension repos**
3. Click **Add**
4. Paste this URL:

```
https://raw.githubusercontent.com/marbou92/MHRepo/main/index.min.json
```

5. Restart Mihon
6. Go to **Browse → Extensions** and install **Comix**

## How This Repo Works

This repository is automatically updated by GitHub Actions when a new release is published from the source repo. It contains:

```
MHRepo/
├── apk/           # Signed extension APKs
├── jar/           # JAR variants
├── icon/          # Extension icons
├── index.min.json # Legacy index (Mihon reads this)
├── index.json     # Full protobuf-as-JSON index
├── index.pb       # Protobuf binary index (Mihon v2)
├── index.html     # Human-readable listing
└── repo.json      # Repo metadata + signing key fingerprint
```

Do NOT edit files here manually — they are generated by CI.
